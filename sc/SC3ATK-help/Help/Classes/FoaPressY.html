            Description

            Apply press to a first order ambisonic signal (B-format) along the y-axis.

            NOTE: FoaPressY is usually called via the convenience wrapper FoaTransform.

            Class Methods

            *ar (w, x, y, z, angle = 0, mul = 1, add = 0)

            From superclass: FoaRotate

            Arguments:

            w

            Input signal W.

            x

            Input signal X.

            y

            Input signal Y.

            z

            Input signal Z.

            angle

            The distortion angle, in radians. -pi/2 to pi/2

            mul

            Output will be multiplied by this value.

            add

            This value will be added to the output.

            Discussion:

            Press is a dominance related transform, specified in terms of a distortion angle. Positive values of angle press the image towards [pi/2, 0] . Negative values press towards [-pi/2, 0] . The default, 0, results in no change.

            NOTE: Corresponding matrix transformer: FoaXformerMatrix: *newPressY
